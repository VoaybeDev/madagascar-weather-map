# 🌍 Carte Météo Madagascar

Carte interactive météo de Madagascar avec données en temps réel.

## Stack
- React + Vite
- Leaflet.js / React-Leaflet
- OpenWeatherMap API
- GeoJSON (GADM)

## Fonctionnalités
- Carte choroplèthe par température
- Marqueurs météo en temps réel
- Sidebar avec détails au clic
- Barre de recherche de ville
- Rafraîchissement automatique toutes les 10 minutes
